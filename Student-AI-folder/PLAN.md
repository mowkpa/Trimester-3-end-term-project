# AI Study Companion - Architecture Plan

## Stack
- React 18 + Vite
- Firebase (Auth + Firestore)
- React Router v6
- Context API
- Gemini API (AI features)
- Recharts (analytics)
- React Hook Form
- Framer Motion (animations)
- Tailwind CSS

## Pages
1. Landing / Auth (Login + Signup)
2. Dashboard
3. Study Planner (CRUD sessions)
4. AI Chat Tutor
5. Flashcards (CRUD)
6. Quiz Generator
7. Progress Analytics
8. Profile Settings

## Core Features
- Auth (Firebase)
- Personalized Dashboard
- Study Session Planner with timer (Pomodoro)
- AI Chat Tutor (Gemini API)
- Flashcard System (spaced repetition)
- Quiz Generator from notes
- Progress Analytics
- Notes CRUD
